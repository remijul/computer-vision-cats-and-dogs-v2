Comment utiliser l'API
  1. Obtenir le token d'authentification : ?C@TS&D0GS!
  2. Envoyer une requête POST à /api/predict
  3. Inclure l'image dans le body et le token dans les headers
  4. Recevoir la prédiction avec le niveau de confiance
Exemple cURL :
curl -X POST "http://localhost:8000/api/predict" \
     -H "Authorization: Bearer ?C@TS&D0GS!" \
     -F "file=@mon_image.jpg"
